## ⚠️ 重要工作流更新 (07:29)

### 微信文章收集工作流（必须遵守）

**用户提醒**: 收到微信公众号文章链接时，**务必使用专用脚本**，不要尝试浏览器抓取！

#### 正确流程
```bash
# 1. 使用脚本抓取并自动归档
python3 skills/link-collector/scripts/archive_wechat.py \
  --url "微信文章链接" \
  --method playwright \
  --auto-classify

# 2. 根据分类结果同步到对应知识库
# - AI文章 → 🤖 AI最新资讯
# - 游戏文章 → 🎮 游戏开发
# - 健康文章 → 🌱 健康生活
```

#### 注意事项
- ❌ 不要用 browser 工具访问微信文章（会触发滑块验证）
- ❌ 不要用 coze-web-fetch（微信会拦截）
- ✅ 必须使用 `fetch_wechat.py` 或 `archive_wechat.py`
- ✅ 使用 `--method playwright` 参数

#### 脚本位置
- `skills/link-collector/scripts/fetch_wechat.py`
- `skills/link-collector/scripts/archive_wechat.py`
