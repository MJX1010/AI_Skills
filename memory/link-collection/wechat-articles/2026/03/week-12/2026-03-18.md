### [Claude Code：把“智商”拉到最高！](https://mp.weixin.qq.com/s/-J479Z14I7yd2jxcwhyLGA)

**摘要**: 如果要发挥 Opus 的全部能力，记得把“智商等级”拉到最高！
今天就来聊一聊它的思考等级。
我之前一直没有留意这个问题，昨天因为一个意外，我去查看模型的时候，才发现有这个功能。
原来 Sonnet 4.6 和 Opus 4.6 都可以调节思考等级。
只要输入
/model
，选中 Opus，然后按键盘的左右方向键就可以切换这个等级了。
经过我的主观感受和客观消息验证，Claude Code 最近调整了这个功能的默认选项。之前的默认选项为 High，而现在的模型选项为 Medium。
所以使用 Claude 的人要注意了，你可能会觉得 Claude 降智了，其实并不是，而是默认的思考等级调低了。
为了更好地来学习和记忆这个功能，我整理一些背景知识。
一、什么是思考等级
Claude Code 引入了一种叫
Effort Level（思考努力等级）
的机制，用来控制模型推理强度。 模型会根据这个等级决定：
是否进行多步推理
推理链条多长
使用多少 token 做内部思考
响应速度 vs 推理质量
本质就是
智力和推理预算控制器
。
